---
title: Trạng thái của hồ sơ
---

Hồ sơ sẽ có một trong ba trạng thái sau:

| Trạng thái | Mô tả                |
| ---------- | -------------------- |
| new        | Hồ sơ mới được tạo   |
| disbursed  | Hồ sơ được giải ngân |
| rejected   | Hồ sơ bị từ chối     |

Dưới đây là hình ảnh mô tả status flow của một hồ sơ
