Hướng dẫn cài đặt Tini App SDK cho dự án React-Native

## Hướng dẫn

#### Cài đặt tiniapp-sdk dependency

- Sử dụng npm

```bash
npm install tiniapp-sdk
```

- Sử dụng yarn

```bash
yarn add tiniapp-sdk
```
