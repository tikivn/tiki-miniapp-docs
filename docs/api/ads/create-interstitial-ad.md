`my.createInterstitialAd` là API dùng để hiển thị quảng cáo dạng popup. Khi dùng jsapi này sẽ trả ra 1 instance [InterstitialAd](interstitial-ad/introduction.md)

**_Khả dụng_**: Hỗ trợ từ version 1.82.1 trở lên.
