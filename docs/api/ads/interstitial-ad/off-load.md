Huỷ việc lắng nghe sự kiện quảng cáo được load thành công.

## Sử dụng

```js
this.ad = my.createInterstitialAd();
this.ad.onLoad(this.onAdLoadHandler);
this.ad.offLoad(this.onAdLoadHandler);
```
