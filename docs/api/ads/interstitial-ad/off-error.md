Huỷ việc lắng nghe sự kiện quảng cáo bị lỗi.

## Sử dụng

```js
this.ad = my.createInterstitialAd();
this.ad.onError(this.onAdErrorHandler);
this.ad.offError(this.onAdErrorHandler);
```
