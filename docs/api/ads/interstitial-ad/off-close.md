Huỷ việc lắng nghe sự kiện quảng cáo bị đóng.

## Sử dụng

```js
this.ad = my.createInterstitialAd();
this.ad.onClose(this.onAdCloseHandler);
this.ad.offClose(this.onAdCloseHandler);
```
