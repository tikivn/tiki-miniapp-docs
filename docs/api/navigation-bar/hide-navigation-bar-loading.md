## Giới thiệu

**my.hideNavigationBarLoading** là API để ẩn đi loading animation trên thanh Navigation Bar của page hiện tại

## Sử dụng

### Sample Code

```js
Page({
  onReady() {
    my.hideNavigationBarLoading();
  },
});
```
