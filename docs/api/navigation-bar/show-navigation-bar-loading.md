## Giới thiệu

**my.showNavigationBarLoading** là API để hiển thị loading animation trên thanh Navigation Bar của page hiện tại

## Sử dụng

### Sample Code

```js
Page({
  onReady() {
    my.showNavigationBarLoading();
  },
});
```
